三角代換（Trigonometric Substitution） 積分技巧

核心思路： 當被積分式含有 \(\sqrt{a^2 - x^2}\)、\(\sqrt{a^2 + x^2}\)、\(\sqrt{x^2 - a^2}\) 的形式時， 用三角函數代換 \(x\)，把根號消去，轉化成三角函數的積分。

情況 ①
含 \(\sqrt{a^2 - x^2}\)
令 \(x = a\sin\theta\)，\(\theta \in [-\frac{\pi}{2}, \frac{\pi}{2}]\)
則 \(\sqrt{a^2-x^2} = a\cos\theta\)，\(dx = a\cos\theta\,d\theta\)
根號消去，得到 \(\cos\theta\) 的積分
應用： \(\displaystyle\int\sqrt{R^2-x^2}\,dx\)（半圓面積）→ #001 半圓等分問題
情況 ②
含 \(\sqrt{a^2 + x^2}\)
令 \(x = a\tan\theta\)，\(\theta \in (-\frac{\pi}{2}, \frac{\pi}{2})\)
則 \(\sqrt{a^2+x^2} = a\sec\theta\)，\(dx = a\sec^2\theta\,d\theta\)
利用恆等式 \(1 + \tan^2\theta = \sec^2\theta\)
情況 ③
含 \(\sqrt{x^2 - a^2}\)
令 \(x = a\sec\theta\)，\(\theta \in [0, \frac{\pi}{2})\cup(\frac{\pi}{2}, \pi]\)
則 \(\sqrt{x^2-a^2} = a\tan\theta\)，\(dx = a\sec\theta\tan\theta\,d\theta\)
利用恆等式 \(\sec^2\theta - 1 = \tan^2\theta\)
推導示範：\(\displaystyle\int\sqrt{R^2 - x^2}\,dx\)
令 \(x = R\sin\theta\)，則 \(dx = R\cos\theta\,d\theta\)，\(\sqrt{R^2-x^2} = R\cos\theta\)
$$\int R\cos\theta \cdot R\cos\theta\,d\theta = R^2\int\cos^2\theta\,d\theta$$ 利用半角公式 \(\cos^2\theta = \dfrac{1+\cos 2\theta}{2}\)： $$= R^2\int\frac{1+\cos 2\theta}{2}d\theta = R^2\left(\frac{\theta}{2} + \frac{\sin 2\theta}{4}\right) + C$$ 用 \(\sin 2\theta = 2\sin\theta\cos\theta\) 與 \(\theta = \arcsin\dfrac{x}{R}\) 換回： $$= \frac{x\sqrt{R^2-x^2}}{2} + \frac{R^2}{2}\arcsin\frac{x}{R} + C$$
不需要背公式——只需記住這個對應關係
圓的方程 \(x^2 + y^2 = R^2\) 中令 \(x = R\sin\theta\) 就得 \(y = R\cos\theta\)。
三角代換的本質就是把代數根號轉換成圓的幾何。
記住這個幾何圖像，三種情況自然推導得出。
