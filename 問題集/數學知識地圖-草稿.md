## 一、數學的整體觀

數學不是一條線，而是一張網。各分支之間相互依賴、彼此滋養。
學習路徑通常從**算術 → 代數 → 分析**這條主幹出發，然後向旁側擴展。

```
【依賴關係總覽】

邏輯學 / 集合論
    │  ← 所有數學的語言基礎
    ▼
算術 ──────────────→ 數論
    │                   │
    ▼                   ▼
代數學 ─────────→ 抽象代數（群論、環論、域論）
    │
    ├──→ 線性代數 ←────────────────────────────┐
    │        │                                 │
    ▼        ▼                                 │
微積分 ──→ 分析學（實分析、複分析、泛函分析）   │
    │        │                                 │
    ├────────┤                                 │
    ▼        ▼                                 │
幾何學 ──→ 拓撲學 ──────────────────────────→─┘
    │
    ├──→ 微分幾何（依賴微積分 + 線性代數）
    │
    ▼
機率論 ──→ 統計學
    │
    ▼
離散數學（組合論、圖論）
    │
    ▼
數值方法 ──→ 計算機科學應用
```

## 二、各分支詳解與相互關係

---

### 1. 邏輯學與集合論（Logic & Set Theory）

**是什麼：**
數學的「語言規則」。定義什麼叫做「證明」、什麼叫做「集合」、什麼叫做「函數」。
所有數學定理的嚴格表述都建立在這套語言之上。

**核心概念：**
- 命題邏輯、謂詞邏輯
- 集合的定義、聯集、交集、補集
- 函數的嚴格定義（定義域、值域、映射）
- 無窮的大小（可數與不可數）、Cantor 對角線論證

**與其他分支的關係：**
- → **所有分支**：提供語言與基礎框架。不懂集合論就沒辦法嚴格定義極限、導函數或向量空間。
- ↔ **代數學**：抽象代數中的「群」、「環」本質上都是帶有結構的集合。
- ↔ **計算機科學**：邏輯閘、布林代數、圖靈機都直接源自數理邏輯。

---

### 2. 算術與初等數學（Arithmetic & Pre-calculus）

**是什麼：**
數的計算規則，以及進入高等數學前的準備知識：代數運算、三角函數、指數對數、函數概念。

**核心概念：**
- 四則運算、因式分解、方程式
- 三角函數及其恆等式（sin、cos、tan 及其反函數）
- 指數函數與對數函數
- 函數的概念（圖形、合成、反函數）

**與其他分支的關係：**
- → **微積分**：極限、導函數需要紮實的函數觀念
- → **數論**：對整數性質的深入研究從算術出發
- 三角函數 → **微積分**中的三角代換、Fourier 分析
- 三角函數 → **幾何學**中的向量角度計算

---

### 3. 代數學（Algebra）

**是什麼：**
研究「運算結構」的數學。從解方程式出發，逐步抽象到任意集合上的運算規律。

#### 3a. 初等代數與線性代數（Elementary & Linear Algebra）

**核心概念：**
- 向量、矩陣的運算
- 線性方程組的解法（高斯消去法）
- 向量空間、線性映射
- 行列式、特徵值與特徵向量

**與其他分支的關係：**
- ↔ **幾何學**：向量是幾何空間的代數語言；旋轉、縮放都是線性變換
- ↔ **分析學**：函數空間本身是向量空間（泛函分析的基礎）
- → **微分方程**：聯立微分方程的解法用到線性代數
- → **機率統計**：共變異數矩陣、主成分分析（PCA）
- → **機器學習**：神經網路的前向傳播即矩陣乘法

#### 3b. 抽象代數（Abstract Algebra）

**核心概念：**
- 群（Group）：只有一種運算的結構（如整數加法）
- 環（Ring）：有加法與乘法的結構（如整數）
- 域（Field）：可以做除法的環（如實數、複數）
- 同態（Homomorphism）、同構（Isomorphism）

**與其他分支的關係：**
- ← **集合論**：群就是帶有運算的集合
- ↔ **數論**：模運算構成有限域，密碼學（RSA、橢圓曲線）的基礎
- ↔ **幾何/拓撲**：對稱群描述幾何物件的對稱性；代數拓撲用群描述空間的「洞」
- ↔ **複分析**：Galois 理論連結域論與多項式的根

---

### 4. 分析學（Analysis）

**是什麼：**
微積分的嚴格化版本。把微積分中直覺性的「趨近」用 ε-δ 語言精確定義，並推廣到更一般的情境。

#### 4a. 實分析（Real Analysis）

**核心概念：**
- 極限的 ε-δ 定義
- 連續性的嚴格定義
- 可微與可積的條件
- 數列與級數的收斂
- Riemann 積分 → Lebesgue 積分（測度論）

**與其他分支的關係：**
- ← **微積分**：實分析是微積分的理論基礎
- → **複分析**：將實數推廣到複數的分析
- → **機率論**：測度論提供機率的嚴格定義（機率 = 測度）
- → **泛函分析**：在函數組成的無窮維空間上做分析

#### 4b. 複分析（Complex Analysis）

**核心概念：**
- 複數的幾何意義（複數平面）
- 解析函數、Cauchy-Riemann 方程
- Cauchy 積分定理、留數定理
- 圍道積分（可用來計算困難的實數積分）

**與其他分支的關係：**
- ← **實分析**：是實分析在複數域的推廣
- ↔ **代數**：代數基本定理（每個多項式都有複數根）的分析證明
- → **物理應用**：量子力學、流體力學廣泛使用複分析

#### 4c. 泛函分析（Functional Analysis）

**核心概念：**
- 巴拿赫空間（Banach Space）、希爾伯特空間（Hilbert Space）
- 線性算子
- 傅立葉分析（Fourier Analysis）

**與其他分支的關係：**
- ← **實分析** + **線性代數**：是兩者在無窮維空間的融合
- → **偏微分方程（PDE）**：大量使用泛函分析工具
- → **量子力學**：量子態就是希爾伯特空間中的向量

---

### 5. 微積分（Calculus）

**是什麼：**
研究連續變化的工具。包含微分（瞬時變化率）與積分（累積量）兩大部分，由微積分基本定理連接。

**核心概念：**
- 極限 → 導函數 → 積分
- 微積分基本定理
- 多變數微積分（偏導數、多重積分）
- 向量微積分（梯度、散度、旋度；Green、Stokes、Gauss 定理）
- 常微分方程（ODE）

**與其他分支的關係：**
- → **分析學**：微積分的直覺提煉成嚴格定義
- ← **三角函數/指對數**：計算中大量依賴
- → **微分幾何**：研究曲面上的微積分
- → **物理**：牛頓力學、電磁學、熱力學的語言
- ↔ **線性代數**：多變數問題（Jacobian、Hessian）連接兩者
- 三角代換 ← **幾何直覺**（見 [#001 半圓等分問題](./001-半圓等分.html)）

---

### 6. 幾何學（Geometry）

**是什麼：**
研究空間與形狀的數學。從歐氏幾何出發，逐步擴展到更抽象的空間。

**核心概念與演進：**
- **歐氏幾何**：平面幾何、畢氏定理、三角形
- **解析幾何**：用座標系研究幾何（代數與幾何的橋樑）
- **微分幾何**：研究曲線與曲面的彎曲程度（曲率、測地線）
- **代數幾何**：用代數方程描述幾何物件

**與其他分支的關係：**
- ↔ **代數**：解析幾何用代數研究幾何；代數幾何反過來用幾何研究代數
- ↔ **微積分**：微分幾何需要微積分計算曲率
- → **拓撲學**：拓撲是幾何的「柔化」版本（只關心連通性，不關心精確形狀）
- → **物理**：廣義相對論的語言是黎曼幾何（微分幾何的一支）

---

### 7. 拓撲學（Topology）

**是什麼：**
研究「連續變形下不變的性質」。一個咖啡杯和甜甜圈在拓撲上是一樣的（都有一個洞）。

**核心概念：**
- 開集、閉集、緊緻性、連通性
- 同胚（兩個空間可以連續雙射）
- 代數拓撲：用代數（群）描述空間的「洞」（基本群、同調群）

**與其他分支的關係：**
- ← **幾何學**：拓撲是幾何的推廣，去掉了距離的概念
- ← **實分析**：分析中的許多概念（連續、緊緻）有拓撲的更一般版本
- ↔ **代數**：代數拓撲用群描述空間結構
- → **物理**：量子場論中的拓撲不變量、材料科學中的拓撲絕緣體

---

### 8. 數論（Number Theory）

**是什麼：**
研究整數（尤其是質數）性質的數學。看似基礎，實則極深。

**核心概念：**
- 質數、因式分解唯一性
- 模算術（Modular Arithmetic）
- 費馬小定理、歐拉定理
- Diophantine 方程（整數解的方程）

**與其他分支的關係：**
- ← **算術**：從整數的基本性質出發
- ↔ **抽象代數**：有限域、Galois 理論與數論深度交叉
- ↔ **分析學**：解析數論用複分析研究質數分布（黎曼ζ函數）
- → **密碼學**：RSA 加密 = 大數因式分解困難（數論直接應用）

---

### 9. 機率論與統計學（Probability & Statistics）

**是什麼：**
- **機率論**：研究隨機現象的數學理論（有嚴格的測度論基礎）
- **統計學**：從資料推斷規律的方法論

**核心概念：**
- 隨機變數、期望值、變異數
- 大數法則、中央極限定理
- 條件機率、Bayes 定理
- 假設檢定、信賴區間
- 隨機過程（Markov chain、布朗運動）

**與其他分支的關係：**
- ← **測度論（實分析）**：機率的嚴格定義
- ← **組合論**：計算離散事件的機率
- ← **線性代數**：共變異數矩陣、多元統計
- → **機器學習**：幾乎所有 ML 模型都建立在統計基礎上
- → **金融數學**：隨機微分方程（Black-Scholes）

---

### 10. 離散數學（Discrete Mathematics）

**是什麼：**
研究「不連續」（離散）結構的數學，是電腦科學的數學基礎。

**核心概念：**
- **組合論**：計算排列、組合；生成函數
- **圖論**：節點與邊的網絡；最短路徑、樹、著色問題
- **數理邏輯**：命題邏輯、布林代數
- **自動機理論**：有限狀態機、圖靈機

**與其他分支的關係：**
- ← **集合論**：離散結構建立在集合語言上
- ↔ **機率論**：隨機圖、隨機演算法
- ↔ **代數**：圖的特徵值（譜圖論）用線性代數研究
- → **計算機科學**：演算法分析、密碼學、網路科學的直接工具

---

### 11. 數值方法（Numerical Methods）

**是什麼：**
研究如何用電腦近似求解解析上無法精確算出的數學問題。

**核心概念：**
- 誤差分析、浮點數
- 數值積分（梯形法、辛普森法）
- Newton 法（方程求根）← 見 [#001 半圓等分問題](./001-半圓等分.html)
- 數值線性代數（LU 分解、奇異值分解）
- 常微分方程的數值解（Runge-Kutta）

**與其他分支的關係：**
- ← **微積分**：對積分、微分方程的近似計算
- ← **線性代數**：大型線性方程組的高效求解
- ← **分析學**：收斂性與誤差界的嚴格證明
- → **工程、物理、機器學習**：幾乎所有計算科學的實作基礎

## 三、學習路徑建議

### 路徑 A：通往純數學

```
邏輯/集合論 → 線性代數 → 實分析 → 複分析
                                  → 拓撲學
                         → 抽象代數 → 代數幾何
                                    → 數論
```

### 路徑 B：通往應用數學 / 工程

```
微積分（單變數 → 多變數 → 向量）
    → 線性代數
    → 常微分方程
    → 偏微分方程
    → 數值方法
    → 機率統計
```

### 路徑 C：通往電腦科學 / 機器學習

```
離散數學（邏輯、圖論、組合）
    → 線性代數
    → 機率統計
    → 微積分（理解梯度下降）
    → 資訊理論
    → 數值方法（選修）
```
